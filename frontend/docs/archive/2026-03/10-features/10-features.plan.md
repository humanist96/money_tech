# MoneyTech - 투자 가치 극대화 10가지 기능 기획

## Context

MoneyTech는 재테크 유튜브/블로그 채널을 데이터 기반으로 분석하여 투자자에게 신뢰할 수 있는 정보를 제공하는 플랫폼.
현재 크롤링, 감성분석, 적중률 추적, NotebookLM 연동 등 기반 인프라가 구축된 상태.
**목표**: 투자자가 실제로 돈을 벌거나 손실을 줄일 수 있는 실용적 기능 10가지 기획.

---

## 10가지 기능 기획안

### 1. 역발상 시그널 (Contrarian Signal)
> "모두가 매수라고 할 때가 고점이다"

- **핵심**: 유튜버 합의도가 극단적(80%+ 매수 또는 매도)일 때 과거 데이터로 역발상 지표 산출
- **구현**: `predictions` + `asset_prices` 테이블 교차 분석
  - 합의도 80%+ 매수 시점 → 이후 1주/1월 수익률 통계
  - 합의도 80%+ 매도 시점 → 이후 실제 반등 확률
- **표시**: "⚠️ 역발상 주의: 삼성전자에 대해 85% 매수 의견 → 과거 유사 상황에서 1개월 후 평균 -3.2%"
- **가치**: 군중심리 반대 투자 전략의 데이터 근거 제공

### 2. 떡상 조기경보 (Buzz Alert System)
> "갑자기 여러 유튜버가 동시에 같은 종목을 언급하면?"

- **핵심**: 24시간 내 5개+ 채널에서 동시 언급되는 종목 실시간 감지
- **구현**: 기존 `buzz_alerts` 테이블 활용 + 실시간 푸시
  - 언급 빈도 급등 감지 (전주 대비 300%+ 증가)
  - 채널 유형별 가중치 (전문 분석가 > 일반 유튜버)
- **알림**: 웹 푸시 / 이메일 / 텔레그램 봇
- **가치**: 시장 움직임을 유튜브 트렌드로 선행 포착

### 3. 유튜버 백테스팅 시뮬레이터
> "이 유튜버 말만 들었다면 내 수익률은?"

- **핵심**: 특정 유튜버의 과거 추천을 그대로 따랐을 때의 가상 수익률 계산
- **구현**: `predictions` 테이블 기반 백테스팅 엔진
  - 초기 투자금 설정 (예: 1,000만원)
  - 유튜버 추천대로 매수/매도 실행
  - 기간별 누적 수익률 차트
  - 벤치마크(KOSPI, BTC) 대비 성과 비교
- **표시**: "삼프로TV 따라하기: 6개월 수익률 +12.3% (KOSPI: +5.1%)"
- **가치**: 유튜버 신뢰도를 수익률로 객관 검증

### 4. 종목 컨센서스 타임라인
> "시간에 따라 유튜버 의견이 어떻게 바뀌었나?"

- **핵심**: 특정 종목에 대한 유튜버별 의견 변화를 시계열로 시각화
- **구현**: `mentioned_assets` + `predictions` 시계열 집계
  - X축: 시간, Y축: 유튜버별 레인
  - 매수(초록)/관망(노랑)/매도(빨강) 색상 표시
  - 가격 차트 오버레이
- **인사이트**: "삼프로TV가 관망→매수로 전환한 시점에 주가 바닥이었다"
- **가치**: 의견 전환 시점이 매매 타이밍의 힌트가 됨

### 5. AI 일일 마켓 브리핑 (오디오 + 텍스트)
> "출근길에 듣는 3분 재테크 요약"

- **핵심**: 전일 업로드된 재테크 영상 핵심 내용을 AI가 종합 요약
- **구현**: NotebookLM 파이프라인 활용 (이미 기반 구축됨)
  - 매일 오전 7시 자동 생성
  - 분야별(주식/코인/부동산) 별도 브리핑
  - 팟캐스트 스타일 오디오 + 텍스트 요약
- **콘텐츠 구조**:
  1. 어제 가장 많이 언급된 종목 TOP 5
  2. 유튜버 간 의견이 갈리는 종목
  3. 신규 추천 종목 및 근거
  4. 오늘 주목할 이벤트
- **가치**: 10개 채널 볼 시간을 3분으로 압축

### 6. 숨은 보석 채널 발굴기
> "구독자는 적지만 적중률 높은 채널"

- **핵심**: 구독자 수 vs 적중률의 역상관 관계에서 숨은 보석 발굴
- **구현**: `channels` 테이블의 `hit_rate` + `subscriber_count` 분석
  - 구독자 < 10만 & 적중률 > 60% 채널 하이라이트
  - 신규 등록 채널 중 초기 성과 우수한 채널 추천
  - 주간 "떠오르는 채널" 랭킹
- **레이더 차트**: 공격성/보수성/다양성/정확도/깊이 5축 시각화
- **가치**: 대형 채널의 후행적 정보 대신, 소형 채널의 선행적 인사이트

### 7. 나만의 유튜버 포트폴리오
> "신뢰하는 유튜버 3~5명의 종합 의견만 모아보기"

- **핵심**: 사용자가 신뢰하는 유튜버를 선택하면 맞춤형 대시보드 생성
- **구현**: 사용자 인증 + 개인화 저장
  - 선택한 유튜버들의 최신 의견 종합 피드
  - 유튜버 조합의 과거 합산 적중률 계산
  - "내 유튜버 포트폴리오 적중률: 64%"
  - 유튜버 간 의견 충돌 시 알림
- **가치**: 정보 과부하 없이 신뢰하는 소스만 추적

### 8. 종목 리스크 스코어보드
> "이 종목에 투자하기 전에 유튜브 여론 체크"

- **핵심**: 특정 종목의 투자 리스크를 유튜버 데이터 기반으로 종합 점수화
- **구현**: 다중 지표 종합 스코어
  - 유튜버 합의도 (매수/매도 비율)
  - 언급 빈도 추이 (급등/급락)
  - 적중률 높은 채널의 의견 가중치
  - 감성 분석 트렌드 (긍정→부정 전환?)
- **표시**: "삼성전자 투자 신호등: 🟡 주의 (Score: 62/100)"
- **가치**: 투자 의사결정 전 빠른 여론 체크

### 9. 주간 적중왕 & 꽝왕 리포트
> "이번 주 가장 잘 맞힌/못 맞힌 유튜버는?"

- **핵심**: 주간 단위로 유튜버 예측 성과 랭킹 + 재미 요소
- **구현**: `predictions` + `asset_prices` 주간 정산
  - 적중왕 TOP 5 (가장 많이 맞힌 유튜버)
  - 꽝왕 TOP 5 (가장 많이 틀린 유튜버) - 유머러스하게
  - 최고의 콜 (가장 수익률 높았던 추천)
  - 최악의 콜 (가장 손실 컸던 추천)
- **공유**: SNS 공유용 카드 이미지 자동 생성
- **가치**: 바이럴 콘텐츠 + 유튜버 책임감 부여 + 재미

### 10. 시장 온도계 (Market Sentiment Gauge)
> "지금 유튜브 재테크 생태계의 분위기는?"

- **핵심**: 전체 유튜버들의 종합 감성을 하나의 지표로 시각화
- **구현**: 전체 `mentioned_assets` 감성 데이터 실시간 집계
  - 탐욕/공포 지수 (0~100, CNN Fear & Greed 스타일)
  - 분야별 온도 (주식 🔥 / 코인 ❄️ / 부동산 🌤️)
  - 과거 극단적 온도 시점 vs 실제 시장 고점/저점 비교
  - "유튜브 탐욕지수 85 → 과거 3번 중 2번은 한 달 내 조정"
- **가치**: 시장 심리를 유튜브 데이터로 계량화한 독자적 지표

## 추천 구현 순서

**Wave 1 (즉시 구현 가능 - 기존 데이터만 활용):**
1. 시장 온도계
2. 주간 적중왕 리포트
3. 숨은 보석 채널

10. 유튜버 포트폴리오
