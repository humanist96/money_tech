## 1. 핵심 방향: 예측형 채널(리딩방) 한정

### 문제 인식
기존 74개 전체 채널 대상 기획은 의미 없음. 적중률/컨센서스/포트폴리오 등은
**매수/매도를 직접 권유하는 채널**에서만 가치가 있다.

### 채널 유형
| 유형 | 코드 | 설명 |
|------|------|------|
| 예측형 | `predictor` | 종목 매수/매도/목표가 직접 제시 |
| 리딩방 | `leader` | 실시간 매매 시그널, 고빈도 리딩 |
| 해설형 | `analyst` | 시장 해설/교육 |
| 미디어 | `media` | 뉴스/보도 |

> **컨텐츠 대상**: `predictor` + `leader`만

## 2. 채널 분류 스코어링 (PIS)

**Prediction Intensity Score** = P1(0.30) + P2(0.25) + P3(0.20) + P4(0.15) + P5(0.10)

| 지표 | 측정 방법 |
|------|----------|
| P1 예측 밀도 | 예측 발생 영상 / 전체 영상 |
| P2 행동 키워드 | 영상당 BUY/SELL 패턴 수 |
| P3 종목 집중도 | TOP5 종목 / 전체 언급 비율 |
| P4 목표가 빈도 | "목표가/XX원" 패턴 영상 비율 |
| P5 감성 편향 | abs(긍정% - 부정%) 평균 |

분류: PIS >= 60 → predictor, 80+ & 고빈도 → leader, 25~60 → analyst, < 25 → media

## 3. 신규 채널 발굴 방법

1. **YouTube 키워드 검색**: "주식 리딩방", "종목 추천", "매수 추천" 등
2. **관련 채널 탐색**: 기존 predictor의 추천/관련 채널
3. **댓글 분석**: 댓글에서 다른 채널 추천 패턴
4. **구독자 필터**: 5K~500K 구독, 월 8+영상, 투자 카테고리

파이프라인: 후보 수집 → 샘플 10개 자막 → 간이 PIS → 전체 PIS → 등록
